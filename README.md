# EduTeach — Modern Education Landing Page

EduTeach is a modern and elegant landing page concept for an educational platform designed for teachers and schools.  
Built using pure HTML, CSS, and JavaScript with a responsive layout, smooth animations, and interactive user experience.

## ✨ Features

- Responsive modern UI
- Elegant dark theme design
- Smooth scrolling navigation
- Animated statistics counter
- Interactive dashboard mockup
- Mobile navigation menu
- Custom cursor animation
- Scroll reveal animations
- Pricing section
- Testimonials section
- CTA subscription form
- Fully built with Vanilla JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
